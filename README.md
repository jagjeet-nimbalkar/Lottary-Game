# 🎰 Lottery Game (React)

A basic lottery game built with **React.js**.

## 🚀 Getting Started

1️⃣ Clone the repo:  
```bash
git clone https://github.com/jagjeet-nimbalkar/Lottary-Game.git
cd Lottary-Game
```

2️⃣ Install dependencies:  
```bash
npm install
```

3️⃣ Run the app:  
```bash
npm run dev
```

## 🛠 Tech Used
- React.js (Vite)
- CSS

## 🎮 How to Play?
- Click **"Generate Numbers"**  
- Random numbers appear  
- Match numbers to win 🎉
